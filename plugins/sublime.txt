1. Для начала скачай установщик sublime'а отсюда: https://www.sublimetext.com/3
2. Установи через установщик sublime. Он сам будет лежать в папке Program Files (x86). Его кэш будет где-то в AppData
3. При запуске у тебя не будет вех необходимых расширений. Тебе понадобятся следующие плагины:
	i. AdvancedNewFile

Остальные плагины установишь по мере необходимости
