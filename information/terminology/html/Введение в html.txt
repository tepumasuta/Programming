HTML  --  расшифровывается, как Hyper Text Markup Language (Язык ГиперТекстовой Разметки)
HTML  --  стандартный язык для создания Веб-страниц
HTML  --  описывает структуру Веб-страницы

Тэг  --  информация, заключённая между <>
Открывающий тэг  --  первый тэг из пары. Выглядит, как <tagname ...>
Закрывающий тэг  --  второй тэг из пары. Выглядит как </tagname ...>
Пустой тэг  --  тэг, не имеющий закрывающего тэга
HTML элемент  --  это всё от открывающего тэга до закрывающего тэга
